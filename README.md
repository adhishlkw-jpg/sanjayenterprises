# Sanjay Enterprises Healthcare Website

A premium medical technology website for Sanjay Enterprises, focused on IVF, fertility, gynaecology and surgical healthcare distribution across India.

## Key Technologies

- Next.js 15 App Router
- React 19
- Tailwind CSS
- Framer Motion
- Lucide React icons
- Next Image remote optimization

## Features

- Sticky navigation and SEO metadata
- Premium healthcare landing page sections
- Product search and category filters
- Product detail panel with manufacturer links
- Partner cards for Hi-Care Plastics, Hope Meditek and Surginova
- WhatsApp quote form integration for `+91 9839075493`
- Responsive layout with scroll and hover animations

## Local Development

Install dependencies and run the Next.js development server:

```bash
npm install
npm run dev
```

Open the local URL shown by Next.js.
